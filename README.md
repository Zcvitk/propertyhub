PropertyHub is a full-stack web application for managing rental properties and bookings.
Users can browse listings, create properties, manage bookings, and leave reviews.

Features:

Authentication (Supabase)
Property listing & management
Booking system
Reviews & ratings
Sorting (price, name, rating)
Dynamic homepage carousel
Image gallery with preview modal
Responsive design

Tech stack:

Frontend: Next.js (App Router)
Styling: Tailwind CSS
Backend/DB: Supabase
Deployment: Vercel

Installation:

npm install
npm run dev

Environment Variables:

Create .env.local and add:
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key

Deployment:

This project is deployed on Vercel.
